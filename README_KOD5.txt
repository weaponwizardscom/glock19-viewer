KOD 5 – pełna szerokość na wszystkich urządzeniach mobilnych (≤ 1024 px)
========================================================================
Ta wersja bazuje na KOD 3 i rozszerza wsparcie mobilne na tablety iPady.

**Pliki do podmiany:**
  • index.html      (media query dla mobile CSS podniesione do 1024 px)  
  • loader.js       (JS breakpoint też 1024 px)  
  • styles.mobile.css (pełna szerokość + drobne poprawki)  

Pozostałe pliki pozostają takie same jak w KOD 3, w tym app.mobile.js.
